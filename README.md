# SocketChat
Многопользовательский чат на websocket с уникальными пользователями
## Команды:
```
git clone https://github.com/frthid/chat_websocket.git
cd chat_websocket
npm run install-all-deps
npm install npm-run-all
npm run dev
```
### Функционал
- Ввод никнейма
- Отправка сообщений
- Сортировка сообщений по времени
